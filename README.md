# POE-programming-assignment-1

## Introduction

##### EasyKanban

A simple task tracker tool written in Java used in conjuction with the Swing framework.

## Prerequisite for running the applcation

- **Need Java**
- **Has to be java 11 or below**
- **You also need Maven**

## How to run the application

- **If within source files**
    - Run **mvn package**
    - **cd target**,
    - **java -jar LoginValidation-1.0-SNAPSHOT.jar**
- **else**
    - **Run the jar file with _java -jar LoginValidation-1.0-SNAPSHOT.jar_**
